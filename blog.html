What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents the structure of a web document as a tree of objects, allowing you to manipulate its content, structure, and style using JavaScript.

What is event delegation in the DOM, and what is it usefull?

Event delegation is a technique in the DOM where you use a single event listener to manage events for multiple elements, instead of adding an event listener to each individual element. This is achieved by taking advantage of event propagation (bubbling), where an event triggered on a child element bubbles up to its parent elements
